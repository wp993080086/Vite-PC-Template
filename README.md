# ⚡️ 简介

一个开箱即用的 `Vite 2` + `Vue 3` + `Element Plus 2` + `Pinia` + `Vue-Router 4` 的PC端项目模板

# 🚀 开发

1. 安装

```
npm install
```

2. 运行

```
npm run serve
```

# 📦️ 多环境打包

```
<!-- 测试环境打包 -->
npm run test_build

<!-- 生产环境打包 -->
npm run build
```

# 🔧 代码检查修复

```
<!-- 格式检查 -->
npm run lint

<!-- 自动修复 -->
npm run lint-fix
```
